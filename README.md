- Google Fonts
- Git and GitHub for version control

### What i learned

This project gave me the opportunity to reinforce my HTML and CSS fundamentals. While recreating the design, I practiced structuring content with semantic HTML and using Flexbox to center elements on the page. I also gained a better understanding of CSS specificity, how to style links and hover states, and how to import custom fonts using Google Fonts. One challenge i faced was getting the card perfectly centered.

### Continued development

Moving forward, i want to continue improving my CSS skills by becoming more confident with Flexbox and learning CSS Grid. I also want to focus on writing cleaner, more maintainable code and building responsive layouts that work well across different screen sizes.

### Useful resources

-Frontend Mentor - Provided the project challenge and design files and even this README file template
-Google fonts - Used to import the Inter font family.

### AI Collaboration

I used ChatGPT as a learning assistant throughout this project. Instead of generating the solution, i used it to clarify HTML and CSS concepts, debug layout issues, understand Flexbox, CSS specificity, hover states, and Google Fonts. This approach helped me understand the reasoning behind the code and strengthened my problem-solving skills.
